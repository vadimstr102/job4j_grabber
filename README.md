# job4j_grabber
The project is a grabber of HTML pages.
#### Functionality:
- Search and load in the database of vacancies from sql.ru by keyword
- Scheduled application launch
- Loading the HTML page with vacancies using simple HTTP Server
#### Technologies:
- JSOUP
- PostgreSQL
- JDBC
- Quartz
- JUnit
